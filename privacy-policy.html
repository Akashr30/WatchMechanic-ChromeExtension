Privacy Policy

WatchMechanic — TimeZone Converter

Last updated: March 28, 2026

Data Collection

This extension does not collect, store, or transmit any personal data, browsing history, or usage analytics. No data is sent to any server or third party.

Local Storage

The extension uses Chrome's built-in chrome.storage.sync API solely to save your timezone preferences (selected timezones, display format, toggle states). This data stays within your Chrome profile and is never accessed by the developer or any external service.

Permissions

  • storage — Used to save and sync your timezone preferences across your Chrome devices.
  • contextMenus — Used to add a "Convert timestamp" option when you right-click selected text.
  • Content script (all URLs) — Required for the double-click timestamp detection feature to work on any webpage. The content script only reads text you double-click; it does not read, scan, or transmit any other page content.

Network Requests

This extension makes zero network requests. It operates entirely offline. All timezone conversions are computed locally using JavaScript's built-in Intl.DateTimeFormat API.

Third-Party Services

This extension does not use any third-party services, analytics tools, tracking pixels, or advertising frameworks.

Changes to This Policy

If this policy is updated, the changes will be reflected on this page with an updated date.

Contact

For questions about this privacy policy, contact the developer through the Chrome Web Store listing.
